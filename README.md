# LynxLocalizations
Repository containing all Lynx 1 &amp; 2 localizations

# Adding your language

Fork this repository, and make a copy of the ```en.lproj``` folder located in ```/LynxLocalizations/Lynx 2/com.mtac.lynxtwo.localizations/Library/PreferenceBundles/lynxprefs.bundle```. Rename this copy to the corresponding language code, for example (French = fr.lproj). A list of language codes can be found [here](https://www.ibabbleon.com/iOS-Language-Codes-ISO-639.html). For each ```.strings``` file, CHANGE ONLY THE LOWERCASE STRING!
